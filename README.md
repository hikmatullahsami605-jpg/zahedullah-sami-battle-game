# 🕹️ Zahedullah Sami Battle Game

This is a simple 2D battle shooting game made using Sketchware.

## 🎮 Game Features
- Player vs Enemy fight
- Health system (HP)
- Fire button for shooting
- Simple battle mechanics

## 🔫 How to Play
- Press FIRE button to attack enemy
- Enemy attacks automatically
- Reduce enemy HP to win
- If player HP becomes 0, game over

## ⚙️ Made With
- Sketchware (Android App Builder)
